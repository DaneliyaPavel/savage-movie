# Доступ к админ-панели

## Проблема

Админ-панель требует аутентификации и прав администратора. При переходе на `/admin` без авторизации происходит редирект на главную страницу.

## Решение

### 1. Создайте администратора

Используйте скрипт для создания администратора:

```bash
./scripts/create-admin.sh admin@example.com your_password
```

### 2. Войдите в систему

1. Перейдите на http://localhost:3000/login
2. Введите email и пароль администратора
3. После входа вы будете перенаправлены на `/admin` (если переходили оттуда)

### 3. Альтернативный способ (для разработки)

Если нужно временно отключить проверку аутентификации для разработки, можно закомментировать проверку в `app/admin/layout.tsx`:

```typescript
// Временно отключено для разработки
// try {
//   const user = await getCurrentUserServer(cookieStore)
//   ...
// } catch {
//   redirect('/')
// }
```

**Внимание:** Не используйте это в production!

## Проверка работы

После входа проверьте:
- http://localhost:3000/admin - должен открыться dashboard
- http://localhost:3000/admin/projects - список проектов
- http://localhost:3000/admin/settings - настройки сайта
