# Описание полей при создании проекта

## Основная информация

### Название (title)

- **Обязательное поле**
- Основное название проекта на языке по умолчанию
- Используется как fallback, если не указаны title_ru или title_en

### Slug

- **Обязательное поле**
- URL-адрес проекта (например: `zarina-project`)
- Генерируется автоматически из названия, но можно изменить
- Должен быть уникальным

### Описание (description)

- Краткое описание проекта на языке по умолчанию
- Используется как fallback, если не указаны description_ru или description_en

### Клиент (client)

- Название клиента или бренда
- Отображается на странице проектов

### Категория (category)

- **Обязательное поле**
- Тип проекта:
  - **Коммерция** - рекламные ролики, коммерческие проекты
  - **ИИ-контент** - проекты с использованием AI
  - **Клипы** - музыкальные клипы
  - **Другое** - остальные проекты

## Видео и изображения

### URL видео (video_url)

- Прямая ссылка на видеофайл (MP4, MOV и т.д.)
- Используется для воспроизведения на странице проекта
- Пример: `https://example.com/video.mp4`

### Ориентация видео (orientation)

- Формат отображения видео в списке проектов
- **Горизонтальное (16:9)** — стандартный формат
- **Вертикальное (9:16)** — Reels/Shorts формат

### Mux Playback ID

- **Важно для главной страницы!**
- ID видео в сервисе Mux
- Используется для отображения видео в carousel на главной странице
- Пример: `Qf6mbMSob4v5nv7c6Mbf7TAipjM01PfHe01bDaDC1otOM`

### URL миниатюры (thumbnail_url)

- **Важно для главной страницы!**
- Ссылка на миниатюру проекта
- Отображается в carousel на главной странице
- Рекомендуемый размер: 16:9

### URL обложки (cover_image_url)

- Ссылка на обложку проекта
- Используется как основное изображение на странице проекта

### Изображения

- Галерея изображений проекта
- Можно загрузить несколько изображений
- Используются для создания thumbnail strip на странице проектов

## Мультиязычность

### Название (RU) / Название (EN)

- Название проекта на русском и английском языках
- Если не указаны, используется основное поле "Название"

### Описание (RU) / Описание (EN)

- Описание проекта на русском и английском языках
- Если не указаны, используется основное поле "Описание"

## Дополнительно

### Год проекта (year)

- Год создания проекта
- Отображается на странице проектов

### Показать на главной странице (is_featured)

- **Важно!**
- Если отмечено, проект будет отображаться в carousel на главной странице
- Рекомендуется отмечать только лучшие/важные проекты (до 6 штук)

### Длительность (duration)

- Длительность видео в секундах
- Используется для отображения информации о проекте

### Роль (role)

- Ваша роль в проекте (режиссер, продюсер, оператор и т.д.)

### Инструменты (tools)

- Список инструментов/программ, использованных в проекте
- Например: "After Effects", "Cinema 4D", "DaVinci Resolve"

### За кадром (behind_scenes)

- Список элементов "за кадром"
- Например: "BTS фото", "Процесс съемки", "Интервью с командой"
