# Руководство по загрузке файлов

## Как загружать изображения для проектов

### Для левого столбца (thumbnail strip)

1. В форме создания/редактирования проекта найдите секцию **"Изображения для левого столбца (thumbnail strip)"**
2. Нажмите "Выбрать файлы" или перетащите изображения в зону загрузки
3. Можно загрузить до 5 изображений
4. Рекомендуется загружать скриншоты/кадры из видео проекта
5. Форматы: JPEG, PNG, WebP
6. Максимальный размер: 10MB на файл

### Где хранятся файлы

- **Локально**: `backend/uploads/images/` и `backend/uploads/videos/`
- **В production**: файлы будут храниться на сервере в той же структуре
- Файлы автоматически получают уникальные имена (UUID)
- URL файлов: `/uploads/images/{filename}` или `/uploads/videos/{filename}`

### Как это работает

1. Файл загружается через API (`/api/upload/images`)
2. Backend сохраняет файл в `backend/uploads/images/`
3. Backend возвращает URL: `/uploads/images/{filename}`
4. Next.js API route (`/api/uploads/[...path]`) отдает файлы из этой директории
5. Изображения доступны по URL `/uploads/images/{filename}`

### Важные моменты

- Файлы **не коммитятся** в git (добавлены в `.gitignore`)
- При развертывании на сервере нужно создать директории `backend/uploads/images/` и `backend/uploads/videos/`
- Для production рекомендуется использовать облачное хранилище (S3, Cloudinary и т.д.)
