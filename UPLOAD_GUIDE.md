# Руководство по загрузке файлов

## Как загружать изображения для проектов

### Для левого столбца (thumbnail strip)

1. В форме создания/редактирования проекта найдите секцию **"Изображения"**
2. Загрузите до 5 изображений для левого столбца
3. Форматы: JPEG, PNG, WebP
4. Максимальный размер: **10MB** на файл

### Галерея проекта

- Все изображения, начиная с 6‑го, отображаются в галерее на детальной странице проекта
- Ограничений по количеству нет (кроме размера файлов)

### За кадром (Behind the Scenes)

- Загружается отдельно в поле "За кадром"
- Можно загружать файлы или вставлять URL

## Загрузка видео

- Форматы: MP4, MOV, AVI
- Максимальный размер: **50MB**
- Видео сохраняются в `backend/uploads/videos`

## Где хранятся файлы

- **Локально**: `backend/uploads/images/` и `backend/uploads/videos/`
- **В production**: та же структура на сервере
- Файлы получают уникальные имена (UUID)

## Как это работает

1. Файл загружается через API backend:
   - `POST /api/upload/image`
   - `POST /api/upload/images`
   - `POST /api/upload/video`
2. Backend сохраняет файл в `backend/uploads/*`
3. Возвращается URL вида `/uploads/images/{filename}`
4. Next.js API route `app/api/uploads/[...path]` отдает файлы

## Важно

- Файлы **не коммитятся** в git
- В Docker загрузки сохраняются в volume `./backend/uploads`
- Для продакшна рекомендуется вынести storage в S3/Cloudinary
